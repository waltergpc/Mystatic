Inventory App

React/MERN Project
Developed in React, API in Node/Express, implements JWT authentication services and product fetching by category.

This MERN app was created using React and react router v6, it sets and inventory for an office environment and list them as general supplies and personal supplies. the front-end was developed using React context API for state management. The api was developed with express where authentication with JWT is implemnented and then routes are provided for the app to get the items for each user, it is lnked to a Mongo database where the users and items are stored.

Bike Terrain Shop

React/MERN Project
Developed in React, API in Node/Express, Auth0 authentication, Stripe Payments, filtering products functionality.

MERN app where a fictional bike store is shown. This app was developed with React and react router v6, also it implements some netlify serverless functions in order to set payments with stripe. Authentication services are implemented through Auth0. Product list and filtering is made throught context APIs. API was developed with Node and express, where a simple API was implemented to supply the stock and specs for each bike, cart functionality connot exceed the numbers of the stock supplied by the API.
